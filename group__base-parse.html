Base function for the API functions.

The API functions take a string or a FILE. This function takes an appropriately initialised Scanner instead. After that it works the same as the text_format_from* family of functions.

Parameters:
descriptora ProtobufCMessageDescriptor of a message you want to deserialise.
scannerA Scanner which will be used by the FSM to parse the text format protobuf.
resultA TextFormatResult instance to record any errors. It is not an option to pass NULL for this and it must be checked for errors.
[in]allocatorAllocator functions.
Returns:
NULL on error. A ProtobufCMessage representation of the text format protobuf on success.
